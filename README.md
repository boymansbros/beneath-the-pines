# Beneath the Pines

Beneath the Pines is a camping-inspired expedition card game built in Unity.

Plan your route, manage your pack, overcome trail challenges, establish camp, and reach your destination before your condition collapses.

Inspired by real backpacking and canoe trips across Ontario, every expedition asks a simple question:

Do you prepare for the trail or prepare for camp?

## Core Gameplay

- Travel between outposts through branching routes
- Resolve trail events using cards
- Manage Daylight, Stamina, Warmth, and Morale
- Build a backpack loadout that fits your strategy
- Complete camp tasks before nightfall
- Adapt to weather, terrain, and unexpected discoveries
- Unlock journal entries, field guides, patches, and future expeditions

## Current MVP Goals

The first playable version focuses on:

- One expedition
- One archetype
- Two in-game days
- Branching trail nodes
- Trail card encounters
- Camp preparation
- Night resolution
- Expedition success or failure

## Technical Goals

- Unity 6
- Data-driven content using ScriptableObjects
- GitHub version control
- Modular systems for expansion
- Solo-developer friendly architecture

## Development Philosophy

Build the smallest playable version first.

Every system must prove its value before becoming more complex.

Priority order:

1. Playable loop
2. Content creation tools
3. Visual polish
4. Expansion systems
